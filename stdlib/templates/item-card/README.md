Description: Item card icon for inventory UIs
Category: items

## Params

- `icon-type` (string, default: "potion") — Icon type: potion, sword, shield, gem, etc.
- `rarity` (string, default: "common") — Rarity border color: common, rare, epic, legendary
- `name` (string, default: "Item") — Item label text
- `width` (number, default: 64) — Card width in pixels
- `height` (number, default: 80) — Card height in pixels

## Output

64x80 PNG icon with colored border based on rarity.
